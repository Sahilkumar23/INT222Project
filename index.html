Movie Booking Website

Developed using HTML, CSS, and JavaScript for the frontend, with Visual Studio Code (VS Code) as the primary development environment, this website offers a seamless and user-friendly experience for movie enthusiasts to browse, select, and book tickets for their favorite films.

Working of an ATM

The "Interactive ATM Simulator" is a Python-based project designed to replicate the functionality of an Automated Teller Machine(ATM) for a single user. Developed using Python programming language and PyCharm as the Integrated Development Environment(IDE), this project offers users the experience of performing basic banking transactions such as withdrawals and deposits from their bank accounts.

This project aims to provide a user-friendly and comprehensive platform for tourists to explore, discover, and book their dream accommodations in this exotic island destination. Developed using HTML, CSS,and JavaScript for the frontend, with Visual Studio Code (VS Code) as the primary development environment, Riviera Retreat promises an unforgettable experience for travelers seeking tranquility, serenity, and natural splendor.
